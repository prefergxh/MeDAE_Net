###### 利用增强型度量去噪自编码器进行开集SEI

参考文献：**Open-Set Specific Emitter Identification Leveraging Enhanced Metric Denoising Autoencoders**

- 问题表述

  本文将整个开集SEI概括为两个主要问题：

  1. 通过最小化经验误差来选取适合的模型

     作者将整个损失函数抽象成以下的公式，把开集识别的首要目标变成选取合适的模型$f$，使得整个损失最小公式如下

     ![image-20260303113158772](./雷达辐射源信号识别学习.assets/image-20260303113158772.png)

     上面的公式包括两个方面分别是交叉熵损失（第一项），它衡量的是在闭集训练时模型预测和标签的交叉熵损失均值。如果整个损失函数只有这一项，那它就是闭集识别。第二项是附加约束损失，用这个损失作为占位来表达开集识别和闭集识别两者的区别，在后面再将它具体化为某个损失函数

  2. 拒识未知信号

     拒识未知信号方面，本文通过比较未知信号与每个类别的中心特征之间距离的阈值来实现。每个类别的中心特征通过文章提出的自编码函数对于类别信号的映射然后取均值得到。公式如下

     ![image-20260304155146673](./雷达辐射源信号识别学习.assets/image-20260304155146673.png)

     如果差值小于阈值$\theta$​则说明可以认为这个未知信号属于已知的类别，否则为未知信号

- 噪声鲁棒性

  噪声鲁棒性指的是模型在面对复杂的噪声干扰下仍能够保持对信号分类的准确度，本文提出了一种噪声正则化的信号重构方法。噪声正则化是之前介绍这个模型的损失函数中附加约束损失的一部分。这里用MSE来代表噪声正规化部分的损失函数，损失函数比较的是原始信号和经过Decoder信号重构后的信号。

  **为什么这样做可以具备噪声鲁棒性**：如果Encoder给出的feature夹带了随机噪声，那么在Decoder重构的信号中，相较于原始信号就会出现毛刺，此时经过MSE损失函数产生的惩罚梯度就会反馈给Encoder和Decoder的网络参数，让其通过学习优化来消除夹带的随机噪声从而实现噪声鲁棒性的功能。实现噪声鲁棒性的模型简图如下：
